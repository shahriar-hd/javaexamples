## ArmStrong
The Armstrong number is a positive m-digit number which is equal to the sum of its m-digit powers, as in the example below : 
<br />1 : 1^1 = 1
<br />2 : 2^1 = 2
<br />153 : 1^3 + 5^3 + 3^3 = 1 + 125 + 27
